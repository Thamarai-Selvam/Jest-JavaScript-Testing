  Node.js  
  Jest Library
 
  
#### Libraries/modules
##### run ```yarn install``` or ```npm install``` inside auth-client dir. to install them  
  yarn/npm  
  jest  
  
    
  
## How to run
Clone this repo  

```
yarn install
yarn test
```
#### For Test Coverage Insights
```
yarn testCov
```
